# 最小栈
主要是维护两个栈，姑且叫原始数据栈和最小值栈，每次“原始数据”要入栈时对应地
将其和最小值栈的栈顶元素比较，谁小变将谁压入最小值栈中，这样最小值栈的栈顶
元素始终时整个栈中最小的。出栈时将两个栈中的栈顶元素同时出栈即可。要获取栈
中的最小值直接拿最小值栈的栈顶元素即可。
